## 🧠 Cognify — *AI Ed-Tech Startup*

> *Not a project. A product.*

**Cognify** is an AI-powered exam prep platform targeting **JEE Main, JEE Advanced, NEET, and BITSAT** — the exams that shape the careers of millions of Indian students every year.

### The Core Loop

```
Learn Concept → Practice Test → Analyze Mistakes → Cogni AI Fixes Gaps → Repeat
      ↑                                                                       |
      └───────────────────────────────────────────────────────────────────────┘
```

### What's Built

| Feature | What it does |
|---|---|
| 🤖 **Cogni AI Tutor** | Groq + Gemini APIs · Socratic method · guides students to answers instead of giving them |
| 📊 **Adaptive Tests** | Tracks accuracy + speed per chapter · auto-generates targeted tests for weak areas |
| 🎯 **Full Exam Simulation** | Real timer · negative marking · PYQ-based questions · exam-specific configs |
| 📈 **Deep Analytics** | Identifies weak chapters, time issues, careless vs conceptual mistakes · actionable recs |
| 👥 **Multi-role System** | Students · Teachers (assign tests) · Admins · Supabase auth throughout |

**Every test result answers one question: _"What went wrong, and how do I fix it?"_**

`Next.js 15` `TypeScript` `Supabase` `Groq API` `Gemini API` `shadcn/ui` `Docker` `Caddy`

🔗 **[View Cognify →](https://github.com/jinnaramuttej/Cognify)**
